# [panda.play](https://panda-play.de)

Codebase for the website of PandaPlay

## Project setup

```bash
git clone https://github.com/devmount/panda.play
cd panda.play
yarn
```

### Compiles and hot-reloads for development

```bash
yarn serve
```

### Compiles and minifies for production

```bash
yarn build
```
